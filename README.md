# Attendance Tracking App (React)

A modern React-based attendance tracking application that enables organizations to monitor employee attendance efficiently through a clean and responsive user interface. The application integrates with the **Attendance Tracking API** to fetch, display, and manage attendance records in real time.

## Features

* Employee attendance dashboard
* Real-time attendance data from the Attendance Tracking API
* View employee check-in and check-out records
* Authentication and authorization
* Responsive and user-friendly interface
* Attendance history and reporting
* Search, filtering, and pagination
* Built with React and TypeScript

## Tech Stack

* React
* TypeScript
* Vite
* React Router
* Axios
* CSS

## Backend

This project consumes data from the **Attendance Tracking API**, a separate ASP.NET Core Web API responsible for authentication, employee management, attendance recording, and data persistence.
